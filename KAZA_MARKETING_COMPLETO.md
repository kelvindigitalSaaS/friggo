# PARTE 1 — DESIGN SYSTEM

## Identidade Visual

| Elemento | Valor |
|----------|-------|
| **Cor primária** | `#0B5449` — Verde profundo |
| **Cor de destaque** | `#22C55E` — Verde claro (CTAs, sucesso) |
| **Cor de alerta** | `#FF7A00` — Laranja (urgência, atenção) |
| **Fundo** | `#F5F5F5` — Branco gelo |
| **Texto principal** | `#0F2F2A` — Verde quase preto |
| **Fundo escuro** | `#0F1713` |
| **Card escuro** | `#1A2420` |
| **Borda** | `#E8ECEB` |

## Tipografia

| Uso | Fonte | Peso |
|-----|-------|------|
| Títulos / Headlines | **Oswald** | Bold 700 |
| Subtítulos | **Oswald** | SemiBold 600 |
| Corpo de texto | **Inter** | Regular 400 |
| Destaques editoriais | **Quattrocento** | Italic |

## Linguagem Visual

- **Cantos arredondados:** `24px` (cards), `16px` (botões)
- **Sombra padrão:** `rgba(11, 84, 73, 0.12) 0px 8px 24px`
- **Gradiente principal:** `linear-gradient(135deg, #0B5449 0%, #165a52 100%)`
- **Gradiente de destaque:** `linear-gradient(135deg, #22C55E 0%, #0B5449 100%)`
- **Estilo:** Clean, minimalista, natureza + tecnologia. Sem excesso de elementos. Muito espaço em branco.
- **Ícones:** Linha fina (stroke), não preenchidos. Estilo moderno.
- **Fotos:** Cenas domésticas reais. Cozinhas modernas, famílias, geladeiras abertas com itens frescos. Luz natural.

## Formatos Instagram

| Formato | Dimensão | Uso |
|---------|----------|-----|
| Feed quadrado | 1080 × 1080px | Posts gerais |
| Carrossel | 1080 × 1080px | Educativo / listas |
| Stories | 1080 × 1920px | CTAs / promoções |
| Reels Cover | 1080 × 1920px | Capa de vídeo |
